Project Overview

Forkify is a dynamic recipe web application. It allows users to search for recipes by name or ingredients and upload their own custom recipes.

The application leverages a third-party API that provides access to a vast collection of recipes. It makes API calls to fetch or submit recipe data. Users can also bookmark their favorite recipes, with these bookmarks stored in the browser's local storage for easy access.

Forkify is built following the MVC (Model-View-Controller) architecture, ensuring a structured and scalable codebase. Additionally, it utilizes asynchronous JavaScript to handle API interactions efficiently, enhancing the overall user experience. Forkify combines powerful functionality with a user-friendly interface, making it a valuable tool for discovering and managing recipes.
